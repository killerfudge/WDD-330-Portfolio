Important New Functions

transform

translate(), translatex(), translate(), scale(), scaleX(), scaleY(), rotate(), skew(), skewX(), skewY()

Notes

the CSS3 transform property allows you to move and alter in appearance any html element with ease. this is done with transform functions.

transforms don’t work on inline elements

scaling defaults to expanding or shrinking around the center, though this can be modified by including the transform-origin property

to add multiple transformations, simple prove a space-separated list of functions. making a second transformation property will override the first

using transform functions only affect the elements they are applied to; other elements don’t readjust to accommodate the changes.

rotate defaults to rotating around the center by a specified angle value that can be expressed in degrees(deg), grads, radians, or turns, though deg are typically used. positive values move clockwise and negative more counterclockwise.

ordering matters, translate() moves along its own axis, so rotating before hand will change where it moves to

transitions all the values of CSS properties to change over time, allowing simple animations

the transition-property defines the CSS properties of the element that should be transitioned, with all for all properties as default.

transitions are declared in a separate area from the other, default, values.

for a property to be transitioned, it needs a valid midpoint.

the -webkit- prefix needs to be used for all transition properties and potentially some of the properties that are being transitioned. This allows the code to support older browsers

the transition-duration property controls how long the transition will take. can be specified in either seconds(s) or milliseconds(ms). 200ms is generally considered the optimum time for a transition.

transition-timing-function allows control over the pace of the changing effect

transition-delay sets time in seconds(s) or milliseconds(ms) before transition begins. setting a negative time causes the transition to begin immediately but partway into the animation

multiple transitions can be included with the same properties, simply by including a comma separated list for each thing we want different in the order they appear in the transition-property tag

the transition sends a transitionend event when it finishes transitioning to the new style or reverting to the old

CSS animations can be used to add in additional points of control called keyframes in our transformations. using them requires creating and naming them, then attaching them to an element by including it in the element’s property declaration block. including a duration is required to make it perceptible

CSS animations are name using the @keyframe notation

animation-direction is used to control whether the animation goes from 0% to 100% or from 100% to 0%
